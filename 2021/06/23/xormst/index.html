CF888G XOR-MST

Posted on 2021-06-23 | Post modified: 2022-07-28 | In Graph Theory
Words count in article: 601 character(s) | Reading time ≈ 3 minute(s)

https://codeforces.com/problemset/problem/888/G?f0a28=1

We are given a complete undirected graph $G$ with $n\ (1\leq n\leq 200000)$ vertices. A number $a_i\ (0\leq a_i< 2^{30})$ is assigned to each vertex, and the weight of an edge between vertices $i$ and $j$ is equal to $a_i\ \mathrm{xor}\ a_j$.

Calculate the weight of the minimum spanning tree in this graph.

In this special MST problem, Boruvka’s algorithm can provide us with new inspiration while other classic methods seems inefficient. To be specific, let’s define set $S_i$ consists of node $i$ only initially. Find the shortest path that connects $i\in S_i$ and $j\in S_j$ and merge sets $S_i$ and $S_i$ repeatedly. A MST can be found eventually.

In this problem, construct a 01-trie $T$ with $a_i$, in which each path from root to leaf resembles a set of merged points. Note that all points with the same value is merged at first. Then search the trie and merge different sets at node $t$ that has both left child $u$ and right child $v$ by enumerate all number $a_i\in u$ and find $a_j\in v$ that generates the lowest weight $w=a_i\ \mathrm{xor}\ a_j$. Add the overall minimum $w$ to our answer each time.

As that a vertex $a\in G$ will only be considered once while enumeration during nodes of it’s path on trie $T$, and that with $a_i$ fixed, it takes $O(\log N)$ to find $a_j$ each time. The overall time complexity should be $O(N\log^2 N)$.

#include<bits/stdc++.h>

constexpr auto MAXN = 200000+5;

struct Node{
    int bits,siz;
    Node *ch[2];
    Node(void){ ch[0] = ch[1] = nullptr; siz = 0; }
    Node(const int& ibits) : bits(ibits) { siz = 0; ch[0] = ch[1] = nullptr; }
}*t0 = new Node(30);

long long num[MAXN];

void Insert(const long long& ins){
    Node* nowAt = t0; long long pos; ++nowAt->siz;
    while(nowAt->bits>=0){
        pos = (ins>>nowAt->bits)&1LL;
        if(nowAt->ch[pos] == nullptr) nowAt->ch[pos] = new Node(nowAt->bits-1);
        nowAt = nowAt->ch[pos]; ++nowAt->siz;
    }
}

long long Query(const long long& val,Node* nowAt){
    if(nowAt->bits<0) return 0LL;
    long long pos = (val>>nowAt->bits)&1LL;
    if(nowAt->ch[pos] != nullptr) return Query(val,nowAt->ch[pos]);
    else return (1LL<<nowAt->bits) + Query(val,nowAt->ch[pos^1]);
}

long long Dfs(Node* nowAt,const int& siz = 0){
    if(nowAt==nullptr || nowAt->bits<0 || nowAt->siz<=1) return 0LL;
    if(nowAt->ch[0]!=nullptr && nowAt->ch[1]!=nullptr){
        long long min = INT64_MAX;
        for(int i=0;i<nowAt->ch[0]->siz;++i)
            min = std::min(min, Query(num[i+siz], nowAt->ch[1]));
        return min + (1LL<<nowAt->bits) + Dfs(nowAt->ch[0],siz) + Dfs(nowAt->ch[1],siz+nowAt->ch[0]->siz); 
    }

    if(nowAt->ch[0] != nullptr) return Dfs(nowAt->ch[0],siz);
    if(nowAt->ch[1] != nullptr) return Dfs(nowAt->ch[1],siz);
    return 0LL;
}

signed main(void)
{
    int totNums;

    std::cin>>totNums;
    for(int i=0;i<totNums;i++) std::cin>>num[i];
    std::sort(num,num+totNums);
    for(int i=0;i<totNums;i++) Insert(num[i]);

    std::cout<<Dfs(t0)<<std::endl;

    return 0;
}
